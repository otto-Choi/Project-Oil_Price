### `garch_model_evaluation.md` — 모델 선택 근거 및 성능 지표

**대상 독자**: 기술 리뷰어, 코드 리뷰어  
**목적**: "왜 GARCH-X를 선택했는가"에 대한 정량적 근거 문서

**주요 내용**:

1. **모델 적합성 지표**: AIC / BIC / Log-Likelihood 비교
   - GARCH(1,1)-t: AIC=38,766 / EGARCH(1,1)-t: AIC=38,737
   - EGARCH가 통계적으로 우수하나 GPR 외생변수 투입 불가 → GARCH 채택

2. **파라미터 유의성**: α+β=0.9916 (안정성 조건 충족), δ=0.000081 (p=0.0016)

3. **GPR 효과 크기**: GPR 80→705 시 밴드폭 변화 +$1.0 (4%) — 실무적 한계 수치화

4. **Coverage Rate 분석**: 이란-미국 전쟁 이탈률 100%와 그 해석

5. **v8 대비 v9 개선 3가지**: Joint MLE / GPR AR(1) 감쇠 / Skewed-t 분포

### `failure_log.md` — 시행착오 전 과정 기록

**대상 독자**: 이 프로젝트의 분석 과정을 이해하고 싶은 모든 사람  
**목적**: 무엇을 시도했고, 왜 실패했으며, 어떤 이유로 방향을 바꿨는지 단계별 기록

> **이 문서가 이 프로젝트에서 가장 중요한 문서다.**  
> 깔끔한 결과물보다 실패를 해부하는 과정이 분석가의 역량을 더 잘 드러낸다.

**기록된 실패 단계 요약**:

| 단계 | 시도 | 핵심 실패 원인 |
|---|---|---|
| EDA | 4-카테고리 분쟁-GPR 상관분석 | 상관계수 0.0~0.24, 예상보다 훨씬 낮음 |
| V1 | 10개 피처 QuantReg + LightGBM | VIF>10 다중공선성, Conflict_Index 중복 정보 |
| V3 | VIF 임계값 5 기반 피처 제거 | 과도한 제거로 유용한 정보까지 손실 |
| V5 | 6개 모델 앙상블 비교 | 비정상성, 보간 누수, GARCH 수렴 실패 |
| V6 | 다중 지평선 예측 (최대 42일) | h=42일에서 GPR 효과 역전 현상 |
| V8 | GARCH-X (분산에 GPR 추가) | γ<0 비경제적 계수, β≥1 분산 폭발 |
| V9 | 한국 소비자 유가 예측 | 유류세 구조적 단절(2022~2026)로 모델 무용화 |

**특히 주목할 섹션**:
- `6단계`: GARCHX 수렴 실패의 3가지 원인 (수치 불안정 / 비경제적 계수 / 외생성 위반)
- `7단계 문제 1`: 한국 유가 분석을 포기한 이유 (정부 개입과 구조적 단절)
- `핵심 실패 원인 정리`: GPR의 역방향 인과 구조 설명
