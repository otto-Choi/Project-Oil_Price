# 시행착오 및 실패 기록

> 이 문서는 프로젝트 전반에 걸친 시행착오, 실패, 방향 전환의 기록이다.
> 무엇을 시도했고, 왜 실패했으며, 어떤 이유로 다른 방안을 택했는지 단계적으로 정리했다.

---

## 프로젝트 개요

**최초 목표**: 중동 분쟁 지수(GPR: Geopolitical Risk Index)와 ACLED 이벤트 데이터를 활용해
Brent 유가를 예측하고, 나아가 **한국 소비자 유가에 미치는 영향**까지 분석

**데이터**: ACLED 분쟁 이벤트 + GPR Index + 유가 데이터 (1996~2026)

---

## 1단계: 중동 분쟁 데이터 EDA (`middle_east_eda.ipynb`)

### 시도한 것
- ACLED 데이터를 주 단위(토요일 기준)로 집계, 2015~2026년
- 분쟁 이벤트를 4개 카테고리로 분류
  - **CAT1**: 고강도 군사 공격 (Armed clash, Air strike, Artillery, Suicide bomb 등) — 269,464건
  - **CAT2**: 비정규 폭력 / 인프라 파괴 (Attack, Explosives, Looting) — 48,383건
  - **CAT3**: 사회 불안 / 시민 불안정 (Protests, Mob violence) — 54,913건
  - **CAT4**: 정치/지정학적 변화 (Territory changes) — 8,772건
- 날짜를 월요일 기준으로 정규화해 주간 GPR 데이터와 정렬
- 이벤트 수 / 사망자 수와 GPR 지표(GPR, GPR_ACT, GPR_THREAT) 간 상관관계 분석

### 상관분석 결과 — 예상보다 훨씬 낮은 수치

| 변수 | GPR_ACT 상관계수 | 해석 |
|------|----------------|------|
| CAT1 Events | **0.244** | 가장 높지만 여전히 약한 양의 상관 |
| CAT1 Fatalities | 0.115 | 매우 약함 |
| CAT2 Events | **-0.013** | 거의 0, 오히려 음수 — 직관과 반대 |
| CAT3 Events | 0.005 | 사실상 상관 없음 |
| CAT4 Events | -0.171 | 음의 상관 — 이해할 수 없는 방향 |

**예상했던 것**: "전쟁이 심해지면 GPR이 오르고 유가도 오른다" → 강한 양의 상관
**실제 결과**: 카테고리별로 상관이 약하거나 심지어 음수

### 왜 이렇게 낮게 나왔나 — 원인 분석

1. **GPR는 뉴스 기반 지수**: 실제 전투 강도가 아니라 뉴스 기사의 키워드 빈도로 산출됨 → 실제 이벤트와 타이밍 미스매치
2. **CAT2, CAT3는 경제와 무관한 이벤트 포함**: 폭동, 시위 등은 유가에 직접 영향 없음
3. **CAT4 (영토 변화)는 음의 상관**: 영토 안정화(전쟁 종료) 시점에 발생 → GPR 하락과 동시 발생
4. **지역 vs 글로벌**: 일부 분쟁은 산유 지역과 무관 (예: 예멘 내전은 GPR은 높이지만 주요 유전과 거리가 있음)

### 해결 시도 및 방향 전환
- 4개 카테고리 → 2개 그룹으로 통합
  - **Group A** (CAT1+CAT2, 군사/폭력): GPR_ACT 상관 0.233 → 그나마 양의 상관
  - **Group B** (CAT3+CAT4, 사회/정치): GPR_ACT 상관 -0.057 → 음의 상관
- 전쟁 기간별 분리 분석
  - 2014~2017 이라크 전쟁: Group A (0.289), Group B (0.165) — 두 그룹 모두 양의 상관
  - 2023~2026 이스라엘-하마스/이란 전쟁: 다른 패턴
- **결론**: 분쟁 이벤트와 GPR/유가의 상관관계는 기대보다 훨씬 낮고 비선형적
  → 단순 상관분석으로는 한계, 모델링 단계로 넘어감

- **문제**: 너무 공격적으로 제거 → 비선형 패턴을 가진 유용한 정보도 날림

## 4단계: 다중 모델 앙상블 비교 (`brent_quantile_forecast_v5.ipynb`)

### 시도한 것

| 모델 그룹 | 모델명 | 특징 |
|-----------|--------|------|
| 통계 모델 | QuantReg | 선형 분위수 회귀 |
| 통계 모델 | GARCH | 조건부 이분산 모델 |
| ML 모델 | LightGBM | 경량 그래디언트 부스팅 |
| ML 모델 | XGBoost | 그래디언트 부스팅 |
| ML 모델 | CatBoost | 범주형 피처 특화 |
| ML 모델 | NGBoost | 확률적 예측 |

- **피처**: GPR_THREAT, GPR_ACT, VIX, DXY, Total_Inventory
- **평가 지표**: Pinball Loss (분위수별 손실)
- **검증**: TimeSeriesSplit (시간 순서 유지)

### 실패들

**문제 1: 비정상성(Non-Stationarity)**
- Brent 유가 수준 자체가 단위근 존재 (ADF 검정 통과 실패)
- 수준(level)으로 예측하면 허구적 회귀 위험
- 해결 시도: 차분(differencing) 또는 로그 수익률로 변환

**문제 2: Total_Inventory 보간 문제**
- 원본: 주간 금요일 발표 데이터
- 일간 모델에 사용하기 위해 선형 보간 시도
- **문제**: 중간 날짜 값이 실제 시장이 알 수 없는 정보를 포함 → 미래 정보 누수 위험

**문제 3: 데이터 누수 (Leakage)**
- h-day 선행 예측 시, lag-1 피처가 t+h-1일 정보 포함 → 엄밀히는 미래 정보
- 각 예측 지평선(horizon)마다 피처를 t 기준으로만 사용하는 규칙 명시화 필요

### 결과
- LightGBM이 가장 우수 (Pinball Loss 기준)
- CatBoost 경쟁력 있음
- GARCH: 수렴 불안정, 별도 버전으로 심화 시도 예정

## 5단계: 이벤트 기반 예측 경로 (`brent_forecast_path_v6.ipynb`)

### 시도한 것
- **다중 지평선 예측**: h = 1, 5, 10, 21, 42일
- **이벤트 기준점**: 주요 지정학적 사건 날짜부터 예측 경로 시각화
  - 러시아-우크라이나 전쟁 (2022-02-24)
  - 이스라엘-하마스 전쟁 (2023-10-07)
  - 이란-이스라엘 충돌 (2024-04-13)
  - 이란-미국 전쟁 (2026-02-28)
- **분위수 밴드**: Q10, Q25, Q50, Q75, Q90
- **GPR_THREAT 연동 밴드 폭**: GPR_THREAT 높을수록 밴드 넓어지는 구조

### 실패: 장기 예측의 역전 현상
- **h=42일 예측**: 때로 높은 위협 → 낮은 가격이라는 역직관적 결과 도출
- **원인**: 분쟁 충격은 단기(1~21일)에만 지속, 이후 수요/공급 펀더멘털로 회귀
- → 장기 예측에서 GPR 효과가 사라지거나 반전됨
- **해결**: 신뢰할 수 있는 예측 지평선을 ≤21일로 제한

### 기타 발견
- Total_Inventory: 예상보다 예측력 약함 (시장은 재고 발표 전에 이미 반응)
- VIX와 GPR_THREAT: 어느 정도 공선관계 → 개별 효과 분리 어려움

## 6단계: GARCHX 모델 — 수렴 실패의 연속 (`brent_garchx_forecast_v8.ipynb`)

### 시도한 것
- **GARCHX 모델**: GARCH(1,1)의 분산 방정식에 외부 변수(GPR_THREAT) 추가
  ```
  수익률 방정식: r_t = μ + φ₁·GPR_ACT + φ₂·GPR_THREAT + φ₃·VIX + φ₄·DXY + ε_t
  분산 방정식:   σ²_t = ω + α·ε²(t-1) + β·σ²(t-1) + γ·GPR_THREAT
  ```
- **동기**: 유가는 변동성 군집 현상이 뚜렷 → GARCH가 이론적으로 적합
- 다양한 사양 시도: GARCH(1,1), GARCH(1,2), EGARCH 등

### 실패들

**문제 1: 수렴 실패**
- BFGS, L-BFGS-B 등 다양한 최적화 알고리즘 시도
- 다수의 시작점(starting values) 시도
- 결과: 빈번한 수렴 실패 또는 수치 불안정

**문제 2: 비경제적 추정 결과**
- 분산 방정식에서 γ < 0 추정 → GPR 높을수록 변동성 감소? — 경제적으로 말이 안 됨
- β ≥ 1 추정 → 변동성 충격이 절대 소멸하지 않음 → 분산 폭발

**문제 3: 외생성 가정 위반**
- GPR_THREAT가 진정한 외생변수가 아님
- 유가 상승 기대 → 언론 과열 → GPR 상승 → 역방향 인과 관계 존재
- → GARCHX의 핵심 가정 위반

**결론**: GARCHX는 이 데이터에서 통계적으로 불안정하고 경제적으로도 해석 불가 → 폐기

## 7단계: 최종 정제 모델 (`upgrade_v9.ipynb`)

### 이전 실패들을 반영한 변경사항

| 변경 항목 | 이전 | 이후 | 이유 |
|-----------|------|------|------|
| 피처 수 | 10개 → 5개 | **4개** (GPR_ACT, GPR_THREAT, VIX, DXY) | 다중공선성 + 불필요한 복잡도 제거 |
| Total_Inventory | 포함 | **제거** | 예측력 약함, 보간 시 정보 누수 위험 |
| 노출 그룹 변수 | 포함 | **제거** | 중복 정보, 다중공선성 주원인 |
| 주 모델 | QuantReg | **LightGBM** | 비선형 효과 포착, 안정적 수렴 |
| GARCH | 포함 | **제거** | 수렴 실패, 외생성 위반 |
| 예측 지평선 | ~42일 | **≤21일** | 장기에서 GPR 효과 반전 |

### 남은 문제들

**문제 1: 한국 유가 분석 실패 (핵심 목표 실패)**
- **원래 목표**: Brent + 분쟁 지수 → 한국 소비자 휘발유 가격 예측
- **시도한 분해**:
  - Brent 원유 (비용의 70~80%)
  - 크랙 스프레드 (정제 마진, 5~15%)
  - 수입 프리미엄 (운임·지리적 요인, 2~5%)
  - 세금·보조금 (정부 정책)
- **실패 원인: 세금 및 유류세 변동 문제**
  - 2022~2026년 한국 정부의 대규모 유류세 인하 및 가격 보조 정책 시행
  - 학습 데이터(2014~2021): 시장 가격
  - 테스트 데이터(2022~2026): 정부 개입으로 통제된 가격
  - → 구조적 단절(Structural Break) 발생, 모델 완전히 실패
  - 세금 컴포넌트를 신뢰성 있게 분리하는 것이 불가능했음
  - 유류세 조정 주기, 인하 폭, 적용 시점이 불규칙하여 패턴화 어려움
- **결론**: 한국 소매 가격 예측은 별도의 정부 정책 모델이 필요 → 현 프로젝트 범위 초과 → **포기**

**문제 2: GPR 데이터 타이밍 지연**
- GPR은 뉴스 아카이브를 주 단위로 집계해서 발표 → 시장 반응보다 항상 늦음
- 시장은 이미 가격에 반영한 뒤에야 GPR이 상승
- → GPR과 유가 변화 간 선행-지연 상관관계가 미약
- 해결 미완: VIX로 일부 대체 시도했으나 GPR 고유 정보를 완전히 대체 못함

## 8단계: 가설 검증 (`0327_가설검증.ipynb`)

### 검증한 가설들

**가설 1: GPR_ACT → 유가 인과 관계**
- 방법: Granger 인과성 검정
- 문제: 양방향 인과 가능성 (GPR ↔ 유가)
  - 전쟁 발발 → 유가 상승 → 언론 과열 → GPR 상승 → 다시 유가 자극 → 루프
- GPR의 외생성(exogeneity) 자체가 의심스러움

**가설 2: 한국 유가 분해**
- 세금 컴포넌트가 가장 큰 불확실성 요인임을 재확인
- 2022~2026 구간: 정부 보조금으로 인해 국제 유가와 국내 가격이 완전히 탈동조화
- **결론**: 이 구간 데이터로는 유의미한 모델 구축 불가

**가설 3: 이벤트 스터디 — 충격 후 가격 반응**
- 분석 이벤트:
  - 2023-10-07 이스라엘-하마스 전쟁: ±5일 윈도우
  - 2024-04-13 이란-이스라엘 충돌: ±5일 윈도우
  - 2026-02-28 이란-미국 전쟁: ±5일 윈도우
- **결과**: 가격 반응은 있었으나 타이밍이 이벤트마다 상이
  - 당일 반응 vs. 2~5일 지연 반응 혼재
  - 규모: $3~15/배럴 수준 (이벤트 심각도에 따라 편차 큼)
  - 장기 지속 여부: 3주 이상 유지된 경우 거의 없음 → 이전 발견과 일치

### 추가 발견: 계절성 미반영
- 수요 계절성(겨울 난방, 여름 드라이빙 시즌) 미포함
- 모델 잔차에서 계절 패턴 관찰됨

## 핵심 실패 원인 정리

### 1. 이벤트-GPR 상관관계가 낮게 나온 이유
- GPR은 뉴스 키워드 기반 → 실제 전투 강도와 시차 존재
- 분쟁 카테고리별로 유가 영향 메커니즘이 다름 (군사 충돌 vs. 시위는 전혀 다른 신호)
- 지역적 분쟁이 글로벌 원유 공급에 영향 안 미치는 경우 많음
- 시장은 GPR 발표 전에 이미 가격 반영

### 2. 한국 유가 분석 실패 이유
- 유류세 인하·보조금 정책이 2022~2026년 동안 불규칙적으로 시행됨
- 세금 구성요소를 분리하는 것 자체가 데이터상 불가능
- 동일 Brent 가격 → 정책에 따라 국내 가격이 전혀 다르게 나타남
- 학습/테스트 데이터 간 구조적 단절 → 모델 무용화
- 결론: 유가 가격 보조 정책을 별도 모델로 다루지 않으면 한국 유가 예측 불가

### 3. GARCHX 실패 이유
- 유가 수익률에 대한 외부 변수의 진정한 외생성 위반
- 수치 최적화 불안정 (분산 방정식에 외부 변수 추가 시 극적으로 악화)
- 경제적으로 해석 불가한 계수 산출 (γ < 0, β ≥ 1)

### 4. 장기 예측 실패 이유
- 분쟁 충격은 1~3주 단기 효과에 그침
- 이후 OPEC 생산량, 미국 셰일 증산, 수요 파괴 등 펀더멘털로 회귀
- → 42일 예측에서 GPR 신호가 무의미해지거나 역전

## 미해결 과제

1. **한국 유가 정책 모델링**: 유류세 인하·보조금을 명시적으로 모델링해야 국내 가격 예측 가능
2. **GPR 외생성 문제**: Granger 검정 결과 양방향 인과 가능성 → 도구변수(IV) 또는 VAR 모델 필요
3. **극단적 사건(Black Swan)**: 2026 이란-미국 전쟁 수준의 사건은 훈련 데이터 분포 밖 → 시나리오 분석으로만 접근 가능
4. **계절성**: 수요 계절성 미반영 → 모델 잔차에 패턴 남음
5. **GPR 타이밍 지연**: 일간 고빈도 대안 지표 필요 (현재로서는 VIX로 부분 대체)
