## 📈 최종 모델: GARCH-X Monte Carlo (v9)

### 모델 구조

```
평균 방정식: r_t = μ + ε_t
분산 방정식: σ²_t = ω + α·ε²_(t-1) + β·σ²_(t-1) + δ·GPR_THREAT_(t-1)
                                                        ↑
                                    GPR이 높을수록 내일 변동성이 수학적으로 커짐
오차항 분포: Hansen(1994) Skewed-t  (ν=6.37, λ=-0.033)
추정 방식:   Joint MLE (7개 파라미터 동시 최적화)
```

### 핵심 파라미터 (모두 p<0.001)

| 파라미터 | 추정값 | 의미 |
|---|---|---|
| α (충격 반응) | 0.0752 | 전일 충격이 오늘 변동성에 미치는 영향 |
| β (변동성 관성) | 0.9164 | 변동성의 자기 지속성 |
| α+β | 0.9916 | 안정성 조건 충족 (<1) |
| δ (GPR 기여) | 0.000081 (p=0.0016) | GPR이 분산에 통계적으로 유의하게 기여 |
| ν (꼬리 두께) | 6.37 | 정규분포 대비 두꺼운 꼬리 확인 |

### v8 대비 v9 개선 사항

| 개선 항목 | v8 | v9 |
|---|---|---|
| delta 추정 | 2-Step OLS (잔차 오염) | Joint MLE (순수 추정) |
| GPR 처리 | 42일 고정 (비현실적 발산) | AR(1) 감쇠 (장기평균 100으로 회귀) |
| 오차 분포 | 대칭 t (하방 과대 추정) | Skewed-t λ=-0.033 (하방 경직성 반영) |

## ⚠️ 모델의 한계와 구조적 실패

이 프로젝트의 핵심 발견은 **예측 성공이 아니라 GPR의 구조적 한계 실증**이다.

| 실패 지표 | 내용 |
|---|---|
| **Granger 인과** | GPR_THREAT → 유가 방향: p=0.59 (비유의). GPR은 선행변수가 아님 |
| **효과 크기** | GPR 80→705 (9배 상승) 시 밴드폭 변화 +$1.0 (4%). 실무적으로 무의미 |
| **이탈률** | 이란-미국 전쟁 Early/Stable Phase 이탈률 100% |
| **GPR 지배 구조** | α+β=0.9916이 변동성의 99.8% 설명. GPR 기여 0.156%에 불과 |

**근본 원인**: GPR은 독립변수가 아닌 역방향 인과 구조를 가짐  
`유가 상승 → 언론 과열 → GPR 상승 → 다시 유가 자극 (순환)`

> 단, 이란-미국 전쟁은 학습 기간 외 신규 레짐(Regime Change)으로, 이탈은 모델 파라미터 실패가 아닌 구조적 한계다.
